What is Matchmaking Bot?

Matchmaking bot is a Discord bot that creates lobbies for ranked competitive play. The ranking system is based off of the elo system used in Chess. Users can queue and match against each other to compete.

What is a lobby?

A lobby is a designated area in the Discord server where users can use the bot. These designated areas are Discord channels. Lobbies are seperated by Discord channels so the implmentor can have several competitions in one channel.
e.g A server may have three lobbies in three seperate Discord channels.
#chess: a channel for chess.
#tetris: a channel for Tetris.
#street-fighter: a channel for Street Fighter.
Each lobby has a seperate queue and ranking system.

How do I use it?

Before the bot can be used in a channel, the channel must first be initialized as a lobby. A server administrator can initialize a lobby with "!f createlobby". After this command is used, the lobby can be used for matchmaking. I strongly suggest having a ruleset for the lobby for users to follow.

Users cannot queue for a match just yet. Before they can queue, users must use "!f register" in an initialized lobby to be able to queue. Each lobby must be registered seperately.

Now, the user is free to queue for lobbies if they are an initialized lobby AND registered by the user. Queuing can be done with "!f queue". The bot will diretly message the user when a match is found. After the bot messages the user, the user may get in contact with the matched user to start a game.

After the match is finished, the loser MUST DIRECTLY MESSAGE THE BOT "!f lose" to report a loss. Only the loser reports the game result. I recommend administrators to take action against those who fail to do so.

For futher help, use "!f" or "!f help".
